# Ford Car Price Prediction

A machine learning project that estimates the resale value of Ford cars. By analyzing a dataset of used car listings, the model identifies key factors that drive vehicle prices, such as mileage, manufacturing year, fuel type, and engine size.

## Key Features

- **Exploratory Data Analysis (EDA)** — Uncovers trends and relationships between car attributes and their market prices using detailed visual plots.
- **Data Processing** — Cleans the dataset and prepares categorical variables (transmission, fuel type) for machine learning algorithms.
- **Predictive Modeling** — Trains a regression model to predict car prices accurately based on historical listing data.

## Model Performance

The regression model achieved an R-squared (R²) score of approximately **0.84** on unseen test data, successfully explaining 84% of the variance in used car prices.
